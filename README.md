# PDF OCR Web Application

## Описание

Веб-приложение для извлечения текста из PDF-файлов с помощью OCR на базе библиотеки EasyOCR. Позволяет пользователю загрузить один PDF-файл (до 50 МБ) и получить результат обработки в виде текстового файла (.txt) и JSON-файла с текстом по страницам. После завершения все временные файлы удаляются для обеспечения конфиденциальности и освобождения ресурсов.

## Функциональные возможности

- **Загрузка PDF-файла** через веб-интерфейс (HTML-форма) с ограничением размера до 50 МБ.
- **Извлечение встроенного текста** напрямую из «цифрового» PDF с помощью PyPDF2.
- **OCR-распознавание** отсканированных страниц: конвертация PDF в изображения (pdf2image) и распознавание текста с помощью EasyOCR.
- **Формат выдачи** результатов:
  - **.txt** — весь распознанный текст одним блоком;
  - **.json** — структура с текстом, разбитым по страницам.
- **Автоматическое удаление** всех временных файлов после завершения обработки.

## Технологии

- **Язык программирования:** Python 3.8+ (Ubuntu 22.04 на Yandex Cloud VM)
- **Бэкенд:** FastAPI + Uvicorn
- **PDF-обработка:** PyPDF2, pdf2image, Pillow
- **OCR:** EasyOCR
- **Шаблонизация:** Jinja2
- **Фронтенд:** HTML, CSS, JS
- **Загрузчик:** python-multipart
